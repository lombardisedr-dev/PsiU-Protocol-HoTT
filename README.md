# Usage and Integration

To use the **PsiU-HoTT** protocol in your own Agda project, follow these steps:

### Installation
1. **Clone the repository** into your local library folder:
   ```bash
   git clone https://github.com
   ```

2. **Register the library**:
   Add the full path to `PsiU-Protocol-HoTT.agda-lib` to your Agda global `libraries` file (usually located at `~/.agda/libraries`).

3. **Declare Dependency**:
   In your project's `.agda-lib` file, add `PsiU-Protocol-HoTT` under the `depend:` field.

### Quick Start
